What to include in your email

1. Service needed: CAD drawing / 3D model / CNC milling / turning / FEA / full package
2. Part description: What is it? What does it do? Any reference dimensions?
3. Material: Steel, aluminium, plastic, etc.
4. CNC controller (for CAM work): e.g. Fanuc, Siemens 840D, Heidenhain, Haas
5. Deadline: When do you need the files?
6. File format: STEP, DXF, PDF, native SolidWorks, etc.
7. Attachments: Sketches, photos, or existing files — anything helps
